About

Posted on Feb 20, 2021

We make a search engine for open source projects!

For many people, GitHub is the only place to go to host, or to look for open source code.
It’s so famous, that for some its even synonymous to to git - a version control system.

As a result, many projects that are not hosted on GitHub are more or less undiscoverable.

With HubGrep, we build a search engine to find all open source projects, equally, no matter where they are hosted.
